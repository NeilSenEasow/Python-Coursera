Usage

To use this script, follow these steps:

1. Ensure you have Python installed on your system.
2. Copy the provided code into a Python script file.
3. Run the script.

Upon running, the script will output the extracted floating-point number representing spam confidence.

Details

The script performs the following steps:

1. It defines a string `text` containing spam confidence information.
2. It finds the position of the floating-point number within the string.
3. It slices the string to extract the floating-point number.
4. It converts the extracted substring to a floating-point number.
5. It prints the extracted floating-point number.

Note

Ensure that the format of the input string remains consistent for accurate extraction.
