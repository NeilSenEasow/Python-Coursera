Usage

To use this script, follow these steps:

1. Ensure you have Python installed on your system.
2. Copy the provided code into a Python script file.
3. Run the script.
4. When prompted, enter the name of the file you want to read.

Upon running, the script will open the specified file, read its content, convert each line to uppercase, and print the modified content.

Details

The script performs the following steps:

1. It prompts the user to enter the name of the file to be read.
2. It opens the specified file in read mode.
3. It reads all lines from the file and stores them in a list.
4. It iterates through each line in the list, removes trailing whitespaces, converts the line to uppercase, and prints it.

Note

Ensure that the specified file exists in the directory from which the script is executed.
